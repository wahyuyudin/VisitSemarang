<English>

This app displays information about Semarang.
There will be 3 categories, they are Kuliner (Culinary),
Wisata (Tourist Atrraction), Kesenian (Traditional Art), Komuntas (Community) and Gedung Pemerintahan (Government Building).
Each categories contain of 5 images and informations that can be read by the user.

It will randomly diplayed the information based on it's category
by pressing the "Berikutnya" button.
It is optimized to run on an Android Nexus 4 or other 4,7" android phone, 
it runs on any device running Android 4.0.3 (Ice Cream Sandwich) or newer.



This project was built on Android Studio 2.2
and Java build 1.8.0_73-b02 amd64
